Privacy Policy

Last updated: March 2026

The short version

Hushh collects no data. It makes no network requests. Nothing you do with Hushh is stored, transmitted, or shared with anyone, including us.

What Hushh does

Hushh is a Chrome extension that blurs sensitive text on web pages during screen recordings. When you click an element or highlight text, Hushh stores that value in the current page's memory and applies a CSS blur filter wherever that value appears in the DOM.

Data collection

Hushh collects no data. Specifically:

  • No secrets, passwords, API keys, or any text you protect is ever read outside the current page.
  • No usage analytics or telemetry of any kind.
  • No crash reports or error logging to any external service.
  • No cookies, no localStorage, no IndexedDB. Protected values exist only in JavaScript memory and are discarded when the tab is closed or navigated away from.

Network requests

Hushh makes zero network requests. The extension operates entirely within your browser. There is no backend, no server, and no external service of any kind.

Permissions

Hushh requests the following Chrome permissions:

  • activeTab: to interact with the currently active tab when you activate the extension.
  • scripting: to inject the content script that applies blur overlays.
  • storage: reserved for future optional preferences (e.g. keyboard shortcut display). No secrets are stored here.

Hushh does not request access to browsing history, bookmarks, or any other sensitive Chrome API.

Third parties

Hushh shares no data with any third party. There are no third-party SDKs, analytics libraries, or trackers included in the extension.

Open source

Hushh is open source. You can inspect the full source code at github.com/Tokonigeorge/hushh to verify these claims.
